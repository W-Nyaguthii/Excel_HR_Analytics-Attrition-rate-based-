# HR Analytics — Employee Attrition Dashboard

An Excel-based data analysis project exploring why employees leave.
---

## Project Overview

This project analyses attrition patterns across 1,470 employees to answer one core question: **what factors drive employees to leave?**

The final deliverable is an interactive Excel dashboard with slicers, KPI cards, and charts — built entirely with native Excel features.

---

## Dataset

- **Source:** [IBM HR Analytics Employee Attrition & Performance](https://www.kaggle.com/datasets/pavansubhasht/ibm-hr-analytics-attrition-dataset) via Kaggle
- **Size:** 1,470 rows × 35 columns
- **Key variable:** `Attrition` (Yes / No)

---

## Tools Used

- Microsoft Excel — Pivot Tables, COUNTIFS, AVERAGEIF, Slicers, Conditional Formatting, Charts

---

## Key Findings

1. **Overall attrition rate is ~16%** — roughly 1 in 6 employees left
2. **Sales has the highest attrition** at nearly 2× the company average
3. **Employees who left earned less** on average than those who stayed
